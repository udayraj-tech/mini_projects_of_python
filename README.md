# 📝 To-Do List (Python)

A simple command-line based To-Do List application built using Python.
This project allows users to manage their daily tasks efficiently with options to add, view, remove, and mark tasks as completed.

---

## 🚀 Features

* ✅ Add multiple tasks
* 📋 View all tasks
* ❌ Remove a specific task
* ✂️ Mark tasks as completed
* ⚠️ Input validation for better user experience

---

## 🛠️ Technologies Used

* Python 3
* List

---

## 📌 How It Works

The program runs in a loop and shows a menu:

1. Add tasks
2. Show tasks
3. Remove a task
4. Mark task as completed
5. Exit

Users can choose an option by entering the corresponding number.

---

## ▶️ How to Run

1. Make sure Python is installed on your system
2. Save the file as `todo.py`
3. Open terminal or command prompt
4. Run the program:

---bash
python todo.py
